# 🌦️ Weather Forecast & Alert Application

## Overview

This project is a Python-based Weather Forecast & Alert System that fetches real-time weather data using public weather APIs and generates alerts for rain, high temperature, humidity, and other weather conditions.

---

## Problem Statement

Users often manually check weather conditions. This project automates weather monitoring and sends useful alerts for planning and safety.

---

## Industry Relevance

Useful for:

* Logistics
* Agriculture
* Event planning
* Travel
* Energy management

---

## Features

* Real-time weather forecast
* Alert generation
* Temperature analysis
* Humidity analysis
* Rain detection
* CSV report generation
* Data visualization
